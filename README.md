# 1-Day Tech Talks Event Application

This project is a simple, single-page web application displaying the schedule for a 1-day technical talks event. It features a dynamic schedule, detailed talk information, and a category-based search functionality.

## Features

*   **Event Schedule:** Displays a clear timeline of all talks and breaks for the day.
*   **Talk Details:** Click on any talk to view its full description.
*   **Category Search:** Filter talks by entering keywords related to their categories.
*   **Responsive Design:** Optimized for viewing on various devices.

## Event Details

*   **Start Time:** 10:00 AM
*   **Talk Duration:** 1 hour per talk
*   **Number of Talks:** 6
*   **Breaks:** 10-minute transitions between talks, and a 1-hour lunch break scheduled after the 3rd talk.

## Technologies Used

*   **Frontend:** HTML, CSS, JavaScript (all inlined for a single-file, serverless deployment)

## How to Run Locally

To view this website on your local machine, follow these simple steps:

1.  **Save the `index.html` file:** Ensure you have the `index.html` file generated by the application in a local directory.
2.  **Open a terminal or command prompt** and navigate to the directory where `index.html` is located.
3.  **Start a simple Python web server** using one of the following commands:
    ```bash
    python3 -m http.server 8000
    # or if python3 is not available
    python -m http.server 8000
    ```
4.  **Open your web browser** and go to `http://localhost:8000`.

You should now see the event schedule and can interact with the search and talk detail features.
